# AskChatApp powered by Socket.io

A basic real-time chat application using Node.js, Express, Socket.io, and HTML/CSS/JS.
Users can join a room and chat together in real time.

## Features

- Real-time messaging (Socket.io)
- Join chat room


Lightweight and easy to extend

## Setup
npm install
npm start


Visit:
http://localhost:3000


 ## Tech Stack

Node.js, Express.js, Socket.io

Socket.io

HTML, CSS, JavaScript
